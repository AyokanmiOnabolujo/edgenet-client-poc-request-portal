# EdgeNet Client PoC Request Portal

A modern web-based business application designed for EdgeNet Consulting to streamline the request, approval, deployment, and tracking of client Proof-of-Concept (PoC) infrastructure resources.

---

## Overview

The EdgeNet Client PoC Request Portal allows Account Managers to request IT infrastructure resources for client PoC engagements while enabling Engineers to manage deployments and monitor request status.

The application provides role-based authentication, request tracking, dashboards, and an intuitive interface for collaboration between business and engineering teams.

---

## Features

- Secure user authentication using Supabase
- Account Manager Dashboard
- Engineer Dashboard
- Create and manage PoC requests
- Request approval workflow
- Deployment status tracking
- Resource allocation
- Activity timeline
- Modern responsive UI
- Notifications and alerts

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Supabase Authentication
- Supabase Database
- Vite
